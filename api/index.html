Readium SDK APIs

This site contains the links to the API documentation for the various public sections of the Readium SDK.

The SDK is composed of three main blocks:

  • Core Library
  • Core Interface Layer
  • DOM Accessor

The Core Library is entirely in C++ but on Android it has a JNI wrapper and for WIndows it has a CLI wrapper.

The Views utility library is JavaScript on all platforms and provides management of the Views and DOM access to the WebKit instance as well as a model with access to the core APIs as needed

Details of the architecture can be found in the Readium SDK Google Docs library.

For more information about Readium and the SDK, please visit Readium.org
